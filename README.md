# ReptileMonitorSys
Pet Monitor System<br>
Temperature and humidity monitoring and control

## Monitoring Part

### Components
<p>Controller: </p>
<blockquote>
  <p>ESP32-WROOM-32</p>
</blockquote><br>

<p>Display: </p>
<blockquote>
  <p> Adafruit SSD1306</p>
</blockquote>

### Each tempareture & humidity set:<br>
<blockquote>
  <p>1 ESP32-WROOM-32</p>
  <p>1 DHT11 sensors</p>
  <p>2 relay</p>
  <p>12V fan</p>
  <p>220V heatpad</p>
</blockquote>

<p>ps.</p>
<p>W1209 input: max 240VAC 5A or 12VDC 10A</p>
